Cost cascade — live examples

One product per logistics tier per brand. Pulled from vendors/<brand>/products in Firestore and recomputed with the current config — so changes you save above reflect here on next refresh. Retail TH includes the 7% customer VAT (on top of the import VAT already inside landed cost). Retail SG converts retail-pre-tax to SGD; 9% GST is applied only when Nubo is flagged as GST-registered.

Formula:
landed_thb = (FOB × EUR/THB × unmatched_landed_uplift) + (CIF × duty_rate) + ((CIF + duty) × thai_vat_rate)
retail_pre_tax_thb = landed_thb ÷ (1 − gross_margin)
retail_th_thb (VAT-incl) = retail_pre_tax_thb × (1 + th_customer_vat_rate)
retail_sg_thb = retail_pre_tax_thb × (1 + sg_customer_gst_rate if Nubo GST-registered else 1)
retail_sg_sgd = retail_sg_thb ÷ (THB/SGD)
Wisdom (China-origin) skips the EU-logistics uplift & tier clamp; uses the import_duty_rate brand override.
