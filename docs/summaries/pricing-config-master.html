1 Architecture Overview

Prices in all three currencies (THB · USD · SGD) are calculated independently from the source-currency FOB cost. retail_usd ≠ retail_thb / FX_USD_THB — each currency is derived from the original cost so rounding errors don't cascade (added v2.31.0).

5 Tax Rules

5a. Thai Import Duty
CIF = fob_thb + freight_thb + insurance_thb duty_thb = CIF × import_duty_rate # By origin: China → 0.00 # ASEAN-China FTA (Form E) EU → 0.10 # Vinci, Berliner, Rampline Korea → 0.10 # DesignPark Norway → 0.10 # Rampline
5b. Thai Import VAT (7%)
vat_thb = (cif_thb + duty_thb) × 0.07 # Paid at customs clearance on CIF+duty value
5c. TH Customer VAT (7%, v2.31.0)
retail_thb = retail_thb_pre_vat × 1.07 # VAT-inclusive retail (TH practice) # Distinct from import VAT above
5d. Singapore GST (9%, deferred)
sg_gst_mult = 1.09 if sg_nubo_gst_registered else 1.0 retail_sgd = retail_pre_tax_thb × sg_gst_mult / (THB/SGD FX) # Currently: sg_nubo_gst_registered = false → zero-rated export, no GST
When Nubo registers for SG GST: set sg_nubo_gst_registered=true in pricing-config, re-run backfill_sgd_pricing.py + sync to Medusa.

6 Per-Brand Formulas

6a. Vinci Play (EU FOB, EUR)
fob_thb = eur_fob × EUR_THB cif_thb = fob_thb + freight_thb + insurance_thb # CBM-based LCL # OR: fob_thb × 1.35 (flat-uplift, no CBM) duty_thb = cif_thb × 0.10 vat_thb = (cif_thb + duty_thb) × 0.07 landed_thb = cif_thb + duty_thb + vat_thb → TIER CLAMP retail_thb = (landed_thb / 0.65) × 1.07 # 35% GM + TH VAT retail_usd = landed_usd / 0.65 # independent calculation retail_sgd = landed_sgd / 0.65 # independent calculation
6b. Berliner (EU EXW, EUR)
eur_fob = list_eur × 0.85 # 15% EXW discount # ... same pipeline as Vinci, with GM = 0.25 (÷ 0.75) retail_thb = (landed_thb / 0.75) × 1.07
6c. DesignPark (Korea FOB, USD)
fob_thb = fob_usd × USD_THB cif_thb = fob_thb × 1.35 # flat-uplift (or CBM when available) duty_thb = cif_thb × 0.10 retail_thb = ((cif_thb + duty + vat) / 0.65) × 1.07
6d. Wisdom / Leka Project (China FOB, USD)
fob_thb = fob_usd × USD_THB cif_thb = fob_thb # China sea: CIF ≈ FOB (or CBM-based) duty_thb = cif_thb × 0.00 # ASEAN-China FTA — 0% (FIXED v2.31.0) vat_thb = cif_thb × 0.07 retail_thb = ((cif_thb + vat) / 0.50) × 1.07 # 50% GM retail_usd = landed_usd / 0.50 retail_sgd = landed_sgd / 0.50
6e. Rampline (Norway EXW, NOK→EUR)
nok_eur = live FX (open.er-api.com / frankfurter.app fallback) eur_fob = net_nok × nok_eur fob_thb = eur_fob × EUR_THB # Path A — airfreight (weight_kg known, 32/127 SKUs as of v2.32.0): freight = weight_kg × 120 THB/kg → cif_thb = fob + freight + insurance # Path B — flat uplift (no weight, 95/127 SKUs): cif_thb = fob_thb × 1.35 duty_thb = cif_thb × 0.10 retail_thb = ((cif_thb + duty + vat) / 0.70) × 1.07 # 30% GM

7 Vinci Tier Floor/Cap System

All EU-origin brands (Vinci, Berliner, Rampline) share this table. It prevents absurdly low landed costs on tiny items (floor) and prevents freight dominating on enormous pieces (cap).

EUR FOB BandFloor (lo_pct)Cap (hi_pct)Floor formulaCap formula
≤ 500 EUR0.802.50fob_thb × 1.80fob_thb × 3.50
≤ 2,000 EUR0.601.80fob_thb × 1.60fob_thb × 2.80
≤ 10,000 EUR0.451.20fob_thb × 1.45fob_thb × 2.20
> 10,000 EUR0.350.80fob_thb × 1.35fob_thb × 1.80
WORKED EXAMPLE — Small item (EUR 150 FOB, Tier 0, Vinci 35% GM)
fob_thb = 150 × 38 = 5,700 THB flat_cif = 5,700 × 1.35 = 7,695 THB duty = 7,695 × 0.10 = 769.5 THB vat = (7,695 + 769.5) × 0.07 = 522.5 THB landed_raw = 7,695 + 769.5 + 522.5 = 8,987 THB Tier 0 floor = 5,700 × 1.80 = 10,260 THB ← raw 8,987 < floor → FLOORED landed_thb = 10,260 THB retail_thb = (10,260 / 0.65) × 1.07 = 16,889 THB ✓

How SGD prices reach Medusa

All 5 brands have retail_sgd in Firestore (backfilled 2026-05-24, v2.31.0). sync_brand_prices_to_medusa.py pushes {"amount": ..., "currency_code": "sgd"} as currency-level price set entries. Medusa matches these to any cart in the Singapore SGD region. No region_id override needed — currency match is sufficient.

Future: When Nubo registers for SG GST → set sg_nubo_gst_registered=true in Firestore → re-run backfill_sgd_pricing.py --brand all --write → re-run sync_brand_prices_to_medusa.py --brand all --write.
